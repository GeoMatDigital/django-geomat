Is it me you're looking for?

Die Entwicklung der Seite findet mit Hilfe von Django (einem Python Web-Framework) statt. Als Datenbank wird PostgreSQL genutzt und die gesamte Installation läuft in einem Docker Container aktuell auf einem Server, der bei DigitalOcean gemietet ist. Der eigentliche Server wird aus einem Rechenzentrum in Frankfurt am Main gestellt der mit Hilfe von PhysikOnline realisiert wurde. Alle hochgeladenen Bilder landen aktuell auf einem kostenlosen Account bei Amazon Web Services und werden für das fertige Produkt auf einen anderen Server kopiert.

Durch die Nutzung eines Docker Containers ist es für die Zukunft möglich das gesamte System ohne größere Probleme auf einen anderen Server zu portieren.

Aktuell wird nach einer Lösung gesucht, um einen eigenen Server innerhalb des Universitätsnetzwerkes zu erhalten. Dazu soll das fertige Produkt unter eine *.uni-frankfurt.de erreichbar sein (z.B. http://geomat.uni-frankfurt.de). Die aktuelle Domain (geomat.baeu.me) ist in Besitz von Michael Gecht und dient zu reinen Testzwecken.

Update, 02. Dezember 2015: Wir sind nun auf einen Server von PhysikOnline umgezogen und sind im Besitz der Domain http://geomat.uni-frankfurt.de!
